xen.windowManager (WindowManager API)

The WindowManager API is used internally in the OS mainly by the System API (xen.system). The window manager contains all the information about the windows, including position, ZIndex (window layer level) and more.

  • xen.windowManager.windows
  • xen.windowManager.addWindow() - (doesn't create a visible window, creates the object with props in the API)
  • xen.windowManager.modifyWindow() - Modify a window's properties within the API, this DOES NOT modify the actual UI, it's for information storage.

xen.system (System API)

  • xen.system.begin() - Initializes the System (Looks for stored Items, saves, etc)
  • xen.system.register(divID, content) - Creates a physical app window on the Desktop
